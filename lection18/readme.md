## 📚 2. Теоретические сведения

### 🔹 Что такое безопасный режим

**Безопасный режим** — это способ загрузки операционной системы, при котором загружаются только самые минимальные компоненты, необходимые для работы системы. В этом режиме не загружаются драйверы сторонних устройств, фоновые приложения и программы автозагрузки.

#### Что можно делать в безопасном режиме:

| Действие | Описание |
|----------|----------|
| **Проверка на вирусы** | Загрузка без автозагрузки позволяет антивирусам работать без блокировок |
| **Восстановление системы** | Запуск средства восстановления в минимальной конфигурации |
| **Установка/обновление драйверов** | Установка драйверов в минимальной среде |
| **Диагностика проблем** | Определение характера проблемы — программная или аппаратная |

#### Признаки:
- Если в безопасном режиме компьютер работает стабильно → проблема в программной части (ПО, драйверы, автозагрузка)
- Если в безопасном режиме проблемы сохраняются → вероятна аппаратная неисправность

---

### 🔹 Точки восстановления системы

**Точка восстановления системы** — это сохранённое состояние системных файлов и настроек Windows на определённый момент времени. Позволяет откатить систему к рабочему состоянию без потери пользовательских файлов.

## ❓ 5. Контрольные вопросы

### 1. Что такое безопасный режим Windows?

**Безопасный режим** — это специальный режим загрузки Windows, при котором загружается только минимальный набор драйверов и служб, необходимых для работы системы (базовые драйверы VGA, мыши, клавиатуры и системные службы). Не загружаются сторонние драйверы, программы автозагрузки и дополнительные службы. Используется для устранения неполадок и диагностики проблем.

---

### 2. Параметры запуска безопасного режима

При загрузке в безопасный режим можно выбрать один из следующих параметров:

| Параметр | Описание |
|----------|----------|
| **Безопасный режим (минимальная)** | Минимальный набор драйверов и служб. Графический интерфейс работает в базовом разрешении VGA. |
| **Безопасный режим (с загрузкой сетевых драйверов)** | То же, что и минимальная, но с загрузкой сетевых драйверов для доступа к сети и интернету. |
| **Безопасный режим (с командной строкой)** | Загрузка только командной строки (без графического интерфейса) с минимальным набором драйверов. |
| **Включить ведение журнала загрузки** | Создание файла `ntbtlog.txt` с информацией о загруженных драйверах и службах. |
| **Включить низкоразрешающий видеорежим (640×480)** | Загрузка с базовым драйвером VGA для исправления проблем с разрешением экрана. |
| **Последняя удачная конфигурация** | Загрузка с параметрами, при которых система успешно загружалась в последний раз. |
| **Режим восстановления служб каталогов** | Только для контроллеров домена Active Directory (восстановление AD). |
| **Режим отладки** | Запуск с режимом отладки для разработчиков и системных администраторов. |

---

### 3. Отличия безопасного режима от чистой загрузки ОС

| Критерий | Безопасный режим | Чистая загрузка |
|----------|------------------|-----------------|
| **Цель** | Диагностика и восстановление системы | Выявление конфликтов между программами |
| **Что загружается** | Только минимальные системные драйверы и службы | Все системные службы Microsoft + минимальные сторонние |
| **Сторонние драйверы** | ❌ Не загружаются | ✅ Загружаются (если не отключены через msconfig) |
| **Автозагрузка** | ❌ Отключена полностью | ❌ Отключена (если настроено через msconfig) |
| **Сетевые драйверы** | По желанию (есть отдельный параметр) | Загружаются (по умолчанию) |
| **Графический интерфейс** | Базовый (640×480 или VGA) | Полнофункциональный |
| **Сторонние службы** | ❌ Не загружаются | ❌ Отключены через msconfig |
| **Использование** | При проблемах с драйверами, вирусах, нестабильной работе | Для устранения конфликтов ПО |
| **Способ вызова** | F8 при загрузке или через msconfig | msconfig → Выборочный запуск |
| **Восстановление системы** | Доступно | Доступно |

**Краткое резюме:**
- **Безопасный режим** — загружает ТОЛЬКО минимальные системные компоненты (почти как новая установка Windows)
- **Чистая загрузка** — загружает ВСЕ системные компоненты Windows, но отключает СТОРОННИЕ службы и автозагрузку
