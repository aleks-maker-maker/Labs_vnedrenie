## 1️⃣ Понятие производительности
**Производительность** — это способность компьютерной системы выполнять определенный объем работы за единицу времени.

### Основные метрики:
| Компонент | Ключевой показатель (Метрика) |
| :--- | :--- |
| **Процессор (CPU)** | Тактовая частота (ГГц), количество операций в секунду (FLOPS) |
| **Видеокарта (GPU)** | Частота кадров (FPS), скорость заполнения пикселей (GPixel/s) |
| **Оперативная память (RAM)** | Пропускная способность (МБ/с), тайминги (CL) |
| **Накопитель (SSD/HDD)** | Скорость последовательного чтения/записи (МБ/с) |

> **Важно:** Производительность зависит не только от "железа", но и от оптимизации драйверов и фоновых процессов ОС.

---

## 2️⃣ Классификация ПО для тестирования (Бенчмарки)

Все утилиты делятся на два больших класса:

### 🔹 Синтетические тесты
Создают абстрактную, максимальную нагрузку на компонент, чтобы выявить его "потолок".
> *Примеры:* **LinX, Prime95, FurMark, 3DMark, CrystalDiskMark.**

### 🔹 Прикладные тесты
Измеряют производительность в реальных задачах (игры, рендеринг, архивация).
> *Примеры:* **Cinebench (рендеринг), HandBrake (кодирование), игровые бенчмарки (Cyberpunk 2077).**
