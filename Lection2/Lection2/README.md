---

## 🔍 Основные понятия

### 1. WinAudit — программа для инвентаризации ПК
> **WinAudit** — это бесплатное приложение для сбора детальной информации об аппаратной и программной конфигурации компьютера.

**Основные возможности:**
- 📌 Сбор данных о процессоре, ОЗУ, дисках, видеокарте
- 📌 Определение установленного ПО и операционной системы
- 📌 Получение сетевых параметров (MAC, IP, DHCP)
- 📌 Экспорт отчета в различные форматы (HTML, PDF, CSV, текст)

---

### 2. Структура отчета WinAudit

После сканирования программа формирует отчет, разделенный на категории:

| Раздел | Содержание |
|--------|------------|
| **1. Обзор системы** | Процессор, ОЗУ, материнская плата, BIOS |
| **2. Операционная система** | Версия ОС, разрядность, установленные обновления |
| **3. Диски** | Модели HDD/SSD, их объем и разделы |
| **4. Видеоадаптер** | Модель видеокарты, объем видеопамяти |
| **5. Сетевые адаптеры** | MAC-адрес, IP-адрес, тип подключения |
| **6. Программы** | Список установленного ПО |
| **7. Пользователи** | Аккаунты, возраст паролей, права доступа |
