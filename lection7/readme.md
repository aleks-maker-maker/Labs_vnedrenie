## 📚 2. Теоретические сведения

### 🔹 Программа AIDA64 Extreme Edition

**AIDA64 Extreme Edition** — мощнейший комплекс диагностики и тестирования Windows-систем, рассчитанный на домашних пользователей. Программа предоставляет широкий спектр инструментов для обзора, помощи в разгоне, диагностики ошибок и проблем, стресс-тестов и мониторинга датчиков системы.

---

### 🔹 Основные возможности AIDA64

| Модуль | Описание |
|--------|----------|
| **Тестирование процессора** | Использует инструкции MMX, SSE, XOP, FMA, AVX и AVX2; поддерживает до 32 потоков |
| **Тестирование памяти и кэша** | Анализ пропускной способности оперативной памяти и задержек |
| **Тестирование дисков** | Определение скорости передачи данных HDD, SSD, оптических и флэш-накопителей |
| **Стресс-тесты** | Максимальная нагрузка на систему для проверки стабильности |
| **Мониторинг датчиков** | 250+ датчиков: температура, напряжение, скорость вентиляторов, энергопотребление |
| **Диагностика** | База из 150 000+ устройств, точные данные о компонентах системы |
| **Анализ ОС и программ** | 50 страниц информации о ПО, лицензиях, процессах, автозагрузке |

### 1. Для чего необходимо выполнять стресс-тесты компьютерной системы?

Стресс-тесты необходимы для:

| Цель | Описание |
|------|----------|
| 🔥 **Проверка стабильности** | Выявление скрытых дефектов и ошибок при максимальной нагрузке |
| 🌡️ **Тестирование охлаждения** | Проверка эффективности системы охлаждения под нагрузкой |
| ⚡ **Выявление "узких мест"** | Определение компонентов, ограничивающих производительность |
| 🔧 **Проверка после разгона** | Оценка стабильности работы после изменения частот и напряжений |
| 🛡️ **Профилактика** | Своевременное обнаружение проблем для предотвращения поломок |
