## 🎯 1. Цель работы

Освоить процесс переноса пользовательских данных и параметров с компьютера под управлением Windows 7 на компьютер с Windows 10 с использованием встроенного средства переноса данных Windows (Windows Easy Transfer) и ручного копирования файлов.

---

## 📚 2. Теоретические сведения

### 🔹 Что такое миграция данных

**Миграция данных** — это процесс переноса пользовательских файлов, документов, настроек приложений и параметров системы со старого компьютера на новый. В Windows для этого используется **Средство переноса данных Windows** (Windows Easy Transfer), которое позволяет автоматизировать процесс переноса.

### 🔹 Способы переноса данных

| Способ | Описание | Преимущества | Недостатки |
|--------|----------|--------------|------------|
| **Средство переноса данных Windows** | Встроенная утилита Windows | Автоматический перенос настроек и файлов | Отсутствует в Windows 10 (нужно копировать из Windows 7) |
| **Ручное копирование** | Копирование папок пользователя через проводник | Простота, полный контроль | Требуется знание расположения папок |
| **Облачные сервисы** | OneDrive, Google Диск, Яндекс.Диск | Доступ с любого устройства | Требуется интернет, ограничение по объёму |
| **Специализированное ПО** | Acronis True Image, EaseUS Todo PCTrans | Профессиональный подход | Платное |

## 🛠️ 3. Выполнение работы

### 🔹 Часть 1. Подготовка данных на Windows 7

#### Шаг 1. Создание файлов для переноса

На компьютере под управлением Windows 7 были созданы следующие объекты:

1. Создана папка с именем **«Для переноса»** (For transferring) в корне диска C:.
2. Создан текстовый файл в Блокноте с текстом **«Со старого ПК»**.
3. Файл сохранён под именем **Data.txt** в папке «Для переноса».
4. Создана папка на Рабочем столе с именем **«Перенос»** (Файлы данных для переноса).

#### Шаг 2. Запуск средства переноса данных Windows

1. Выбран путь: **Пуск** → **Все программы** → **Стандартные** → **Служебные** → **Средство переноса данных Windows**.
2. Открылось окно приветствия, нажата кнопка **Далее**.
3. В окне «Выберите способ переноса файлов и параметров на новый компьютер» выбран пункт **«Внешний диск или USB-устройство флэш-памяти»**.

#### Шаг 3. Выбор источника данных

1. В окне «Какой компьютер сейчас используется?» выбран пункт **«Это мой исходный компьютер»**.
2. Произведена проверка возможности переноса данных.

#### Шаг 4. Выбор данных для переноса

1. В окне «Выберите данные, переносимые с этого компьютера»:
   - Сняты флажки со всех учётных записей, кроме текущей.
   - Для текущей учётной записи нажата кнопка **«Настройка»**.
2. В окне настройки учётной записи нажата кнопка **«Дополнительно»**.
3. Найдена папка «Для переноса» и выбран файл **Data.txt**.
4. Нажата кнопка **«Сохранить»**.

**Размер переносимого файла:** 1,5 КБ.

#### Шаг 5. Сохранение файла переноса

1. Нажата кнопка **Далее**.
2. Появилось окно «Сохранение файлов и параметров для переноса» (пароль не вводился).
3. Нажата кнопка **«Сохранить»**.
4. В качестве места сохранения выбрана папка «Перенос» на Рабочем столе.
5. Дождались окончания процесса сохранения.
6. Нажата кнопка **Далее**, затем **Закрыть**.

### 🔹 Часть 2. Подготовка Windows 10

#### Шаг 6. Установка утилит виртуальной машины

1. Запущена виртуальная машина с Windows 10.
2. В меню виртуальной машины выбран пункт **«Установить дополнения гостевой ОС»**.
3. Запущен установочный файл дополнений.
4. Выполнена установка дополнений и перезагрузка виртуальной машины.

#### Шаг 7. Копирование папки «Перенос»

1. В Windows 10 подключён общий доступ к диску Windows 7.
2. Папка «Перенос» скопирована из Windows 7 в Windows 10 на Рабочий стол.

---

### 🔹 Часть 3. Перенос данных с помощью migwiz

#### Шаг 8. Копирование средства переноса

Так как в Windows 10 средство переноса данных отсутствует, оно было скопировано из Windows 7:

1. На Windows 7 найдена папка `C:\Windows\System32\migwiz`.
2. Папка `migwiz` скопирована на Рабочий стол Windows 10.

#### Шаг 9. Запуск средства переноса на Windows 10

1. В папке `migwiz` на Windows 10 запущен файл `migwiz.exe`.
2. Средство переноса данных Windows запущено.
3. Нажат пункт **«Продолжить»**.

#### Шаг 10. Выбор файла переноса

1. Выбран пункт **«Внешний диск или USB-устройство флэш-памяти»**.
2. Выбран пункт **«Это мой новый компьютер»**.
3. В окне «Где хранятся файлы переноса?» выбран пункт **«Внешний диск или USB-устройство флэш-памяти»**.
4. Указан путь к файлу переноса (папка «Перенос» на Рабочем столе).

#### Шаг 11. Выбор данных для восстановления

1. В окне «Выберите данные, переносимые на этот компьютер» нажата кнопка **«Настройка»**.
2. В окне настройки учётной записи выбран файл `Data.txt` для переноса.
3. Нажата кнопка **«Сохранить»**.
4. Нажата кнопка **«Перенести»**.

#### Шаг 12. Завершение переноса

1. Начался процесс переноса данных.
2. После завершения переноса появилось окно с сообщением об успешном переносе.
3. Нажата кнопка **«Готово»**.
4. Выполнена перезагрузка компьютера.

### 2. Какие способы миграции (переноса) данных вы можете назвать?

| Способ | Описание |
|--------|----------|
| **Средство переноса данных Windows** | Встроенная утилита (в Windows 7) и через migwiz (в Windows 10) |
| **Ручное копирование** | Копирование папок пользователя (Документы, Загрузки, Рабочий стол) через проводник |
| **Облачные сервисы** | OneDrive, Google Диск, Яндекс.Диск, Dropbox |
| **Внешние накопители** | Копирование на USB-флешку, внешний жёсткий диск |
| **Специализированное ПО** | Acronis True Image, EaseUS Todo PCTrans, Zinstall WinWin |
| **Сетевое копирование** | Через локальную сеть (SMB-общие папки) |
