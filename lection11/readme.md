## 📚 2. Теоретические сведения

### 🔹 Что такое «чистая загрузка»

**Чистая загрузка** — это режим запуска Windows с минимальным набором драйверов и программ. Он позволяет исключить конфликты между приложениями и службами, которые могут мешать нормальной работе системы.

#### Когда применяется чистая загрузка:
- 🔹 Устранение ошибок при установке обновлений
- 🔹 Выявление конфликтующего ПО
- 🔹 Диагностика проблем с производительностью
- 🔹 Устранение ошибок, возникающих случайным образом

---

### 🔹 Команда SFC (System File Checker)

**SFC** — утилита командной строки, предназначенная для проверки целостности и восстановления повреждённых системных файлов Windows.

#### Основные параметры команды SFC:

| Параметр | Описание |
|----------|----------|
| `/scannow` | Проверка всех защищённых системных файлов и восстановление повреждённых |
| `/verifyonly` | Проверка целостности системных файлов без их восстановления |
| `/scanfile=путь` | Проверка указанного файла с попыткой восстановления |
| `/verifyfile=путь` | Проверка указанного файла без восстановления |
| `/offbootdir=путь` | Указывает расположение загрузочного раздела (для офлайн-сканирования) |
| `/offwindir=путь` | Указывает расположение системной папки (для офлайн-сканирования) |

## 🛠️ 3. Выполнение работы

### 🔹 Часть 1. Выполнение чистой загрузки

#### Шаг 1. Запуск оснастки «Конфигурация системы»

1. Нажата комбинация клавиш `Win + R`
2. В поле «Выполнить» введена команда: `msconfig`
3. Нажата клавиша `Enter`

![Скриншот 1 - Запуск msconfig](img/msconfig-run.png)
*Рисунок 1 – Запуск оснастки «Конфигурация системы»*

---

#### Шаг 2. Настройка выборочного запуска

1. На вкладке **Общие** выбран параметр **«Выборочный запуск»**
2. Снят флажок **«Загружать элементы автозагрузки»**

![Скриншот 2 - Настройка выборочного запуска](img/msconfig-general.png)
*Рисунок 2 – Вкладка «Общие» с выбранным выборочным запуском*

---

#### Шаг 3. Отключение сторонних служб

1. На вкладке **Службы** установлен флажок **«Не отображать службы Майкрософт»**
2. Нажата кнопка **«Отключить все»**

![Скриншот 3 - Отключение сторонних служб](img/msconfig-services.png)
*Рисунок 3 – Отключение служб, не относящихся к Microsoft*

3. Нажата кнопка **ОК**
4. В появившемся диалоговом окне нажата кнопка **«Перезагрузка»**

---

### 🔹 Часть 2. Проверка системных файлов Windows

#### Шаг 4. Запуск командной строки от имени администратора

1. Открыто меню **Пуск**
2. Найден пункт **«Командная строка»**
3. Нажата правая кнопка мыши, выбран пункт **«Запуск от имени администратора»**

![Скриншот 4 - Запуск командной строки от имени администратора](img/cmd-admin.png)
*Рисунок 4 – Запуск командной строки с правами администратора*

---

#### Шаг 5. Выполнение проверки системных файлов

1. В командной строке введена команда: `sfc /scannow`
2. Нажата клавиша `Enter`
