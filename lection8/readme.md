## 📚 2. Теоретические сведения

### 🔹 Программа PassMark PerformanceTest

**PerformanceTest** — набор тестов, позволяющих оценить общую производительность вашего ПК по сравнению с другими компьютерами. В программу входят двадцать семь стандартных тестов в семи группах.

#### Задачи утилиты:
- 🔹 Выяснение того, работает ли ПК на должном уровне
- 🔹 Сравнение производительности с машинами подобной конфигурации
- 🔹 Объективные измерения для принятия решений при покупке

#### Стандартные наборы тестов:

| Группа тестов | Что тестируется |
|---------------|-----------------|
| **CPU** | Математические операции, сжатие, шифрование, MMX/SSE, 3DNow! |
| **2D Graphics** | Черчение, битовые карты, шрифты, текст, GUI-элементы |
| **3D Graphics** | DirectX 8.1, DirectX 9, анимация |
| **Memory** | Скорость доступа к памяти, эффективность |
| **Disk** | Чтение, запись и поиск файлов |
| **CD/DVD** | Скорость оптических драйверов |
