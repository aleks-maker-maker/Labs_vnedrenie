## 🛠️ Стек технологий

| Инструмент | Назначение |
|------------|------------|
| **CrystalDiskMark** | Тестирование скорости чтения/записи дисков |
| **CrystalDiskInfo** | Мониторинг состояния дисков (S.M.A.R.T.) |
| **PassMark BurnInTest** | Нагрузочное тестирование ПК («прогонка») |
| **PC Wizard** | Диагностика и тестирование компонентов ПК |
| **dxdiag** | Средство диагностики DirectX |
