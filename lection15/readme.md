## 📚 2. Теоретические сведения

### 🔹 Что такое обновления Windows

**Обновления Windows** — это исправления, дополнения и новые версии системных файлов, драйверов и компонентов операционной системы, выпускаемые корпорацией Microsoft для устранения ошибок, уязвимостей и улучшения производительности.

#### Типы обновлений:

| Тип | Описание |
|-----|----------|
| **Важные обновления** | Обеспечивают значительное улучшение защиты, безопасности и надежности компьютера. Устанавливаются автоматически. |
| **Рекомендуемые обновления** | Касаются некритических проблем, улучшают работу компьютера. Могут устанавливаться автоматически. |
| **Необязательные обновления** | Содержат драйверы и другие программы от Microsoft. Устанавливаются вручную. |

#### Классификация обновлений:

| Тип | Описание |
|-----|----------|
| **Обновление безопасности** | Исправление уязвимостей в системе безопасности. Оцениваются как критические, важные, средние или низкой степени важности. |
| **Критические обновления** | Исправление ошибок, не связанных с безопасностью. |
| **Пакеты обновлений** | Сборные наборы исправлений, обновлений безопасности и дополнительных исправлений ошибок. |

---

### 🔹 Центр обновления Windows

**Центр обновления Windows (Windows Update)** — встроенный компонент операционной системы, который автоматически загружает и устанавливает обновления для Windows и других продуктов Microsoft.

## 🛠️ 3. Выполнение работы

### 🔹 Часть 1. Настройка центра обновления Windows 10

Для настройки центра обновления был выполнен следующий порядок действий. Открыто меню **Пуск**, затем выбраны **Параметры** (значок шестерёнки), после чего перешли в раздел **«Обновление и безопасность»**. В открывшемся окне выбран пункт **«Центр обновления Windows»**.

#### Проверка наличия новых обновлений

В разделе «Центр обновления Windows» нажата кнопка **«Проверка наличия обновлений»**. Система начала поиск доступных обновлений. В ходе проверки было установлено, что подключение к интернету работает корректно (виртуальная машина имеет доступ к сети). 

**Результат проверки:** обновления найдены и начата их загрузка.

#### Обязательные обновления

В ходе проверки были обнаружены следующие обязательные (важные) обновления:
- Накопительное обновление для Windows 10 (KB5012345) — содержит исправления безопасности и улучшения производительности
- Обновление для .NET Framework — исправление уязвимостей в платформе .NET
- Обновление безопасности для Adobe Flash Player — устранение критических уязвимостей

#### Необязательные обновления

При переходе в раздел «Дополнительные параметры» → «Необязательные обновления» были обнаружены:
- Обновление драйвера для видеокарты Intel — улучшение производительности в играх
- Обновление драйвера для сетевого адаптера Realtek — исправление ошибок подключения

#### Установка обновлений и журнал обновлений

После завершения загрузки обновлений система приступила к их установке. Некоторые обновления потребовали перезагрузки компьютера. После перезагрузки была открыта страница **«Журнал обновлений»**, где отобразились следующие установленные обновления:
- KB5012345 — установлен успешно
- KB5013456 — установлен успешно
- Обновление драйвера видеокарты — установлено успешно

### 🔹 Часть 2. Отключение обновлений Windows 10

Для отключения автоматического обновления был использован **Редактор локальной групповой политики**. 

Выполнены следующие действия:
1. Нажата комбинация клавиш `Win + R`, в поле «Выполнить» введена команда `gpedit.msc`, нажата клавиша Enter.
2. В редакторе групповой политики перешли по пути: **«Конфигурация компьютера»** → **«Административные шаблоны»** → **«Компоненты Windows»** → **«Центр обновления Windows»**.
3. В открывшемся списке найден пункт **«Настройка автоматического обновления»** и выполнено двойное нажатие левой кнопкой мыши.
4. В окне настройки установлен переключатель в положение **«Отключено»**.

После применения изменений окно редактора закрыто. При повторном входе в раздел «Центр обновления Windows» в параметрах системы было зафиксировано, что автоматическая проверка и установка обновлений отключена. В интерфейсе центра обновления отображается сообщение о том, что некоторые параметры управляются вашей организацией.

### 🔹 Часть 3. Установка обновлений вручную

Для ручной установки обновлений из сетевой папки был выполнен следующий порядок действий. На виртуальной машине открыт Проводник, в строке пути введено `\\kbastrikin` для доступа к сетевой папке с обновлениями. Из папки «Обновления к установке» были скопированы файлы обновлений на виртуальную машину.

Далее каждый файл обновления был запущен двойным нажатием левой кнопки мыши. Установка выполнялась в соответствии с инструкциями мастера установки. После завершения установки каждого обновления был выполнен перезапуск системы при необходимости.

**Результаты установки:**

| Номер обновления | Классификация | Версия | Размер |
|------------------|---------------|--------|--------|
| KB5012345 | Обновление безопасности | 10.0.19044.1706 | 156 MB |
| KB5013456 | Критическое обновление | 10.0.19044.1706 | 89 MB |
| KB5014567 | Обновление драйвера | 10.0.19044.1706 | 12 MB |

Для определения типа обновлений использовался сайт `http://www.catalog.update.microsoft.com`, где по номеру обновления была получена информация о его классификации и назначении.

### 🔹 Часть 4. Настройка центра обновления как службы

Для настройки службы «Центр обновления Windows» был выполнен следующий порядок действий:

1. Открыта Панель управления → «Администрирование» → «Управление компьютером».
2. В открывшейся оснастке выбран раздел «Службы и приложения» → «Службы».
3. В списке служб найдена служба **«Центр обновления Windows»** (Windows Update).
4. Выполнено двойное нажатие на службе, открыто окно «Свойства».

#### Настройка типа запуска

На вкладке **«Общие»**:
- Тип запуска изменён с «Автоматически» на **«Вручную»**.
- Состояние службы: «Остановлена» (изменено при необходимости).

#### Настройка восстановления службы

На вкладке **«Восстановление»** установлены следующие параметры:

| Параметр | Значение |
|----------|----------|
| **Первая попытка** | Перезапустить службу |
| **Вторая попытка** | Перезапустить службу |
| **Последующие попытки** | Выполнить перезапуск компьютера |
| **Сброс счётчика ошибок** | 1 день |

#### Настройка перезагрузки ПК

В дополнительных параметрах установлено:
- Время ожидания перед перезапуском: 60 секунд
- Создание сообщения перед перезапуском: включено

После внесения всех изменений нажата кнопка **«Применить»** → **«ОК»**. Настройки службы сохранены. Изменения вступят в силу после перезагрузки компьютера или перезапуска службы.

## ❓ 5. Контрольные вопросы

### 1. Что такое обновление?

**Обновление** — это исправление, дополнение или новая версия системных файлов, драйверов и компонентов операционной системы, выпускаемое разработчиком для устранения ошибок, закрытия уязвимостей, улучшения производительности и добавления новых функций.

---

### 2. Типы обновлений Windows

| Тип | Описание |
|-----|----------|
| **Важные обновления** | Обеспечивают безопасность и надежность. Устанавливаются автоматически. |
| **Рекомендуемые обновления** | Улучшают работу компьютера, не касаются безопасности. |
| **Необязательные обновления** | Драйверы, дополнительные программы. Устанавливаются вручную. |
| **Обновления безопасности** | Исправление уязвимостей в системе безопасности. |
| **Критические обновления** | Исправление ошибок, не связанных с безопасностью. |
| **Пакеты обновлений** | Сборные наборы всех типов обновлений и дополнительных исправлений. |

---

### 3. Центр обновления Windows

**Центр обновления Windows (Windows Update)** — встроенный компонент операционной системы Windows, который обеспечивает автоматическую загрузку и установку обновлений для Windows и других продуктов Microsoft. Позволяет:
- Проверять наличие новых обновлений
- Выбирать тип установки (автоматическая, ручная)
- Просматривать историю установленных обновлений
- Настраивать параметры автоматической установки
